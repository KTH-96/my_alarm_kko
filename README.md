## MY Alarm v1.0.0
____
### 서비스 소개
하루에 필요한 일들을 카카오톡 메세지로 지정 시간에 보내주는 프로젝트

### 사용 기술 및 환경
- Java 11
- spring boot version 2.7.6
- gradle 7.5.1
- Selenium
